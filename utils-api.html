as-str

  function

  
Usage: (as-str x)


  
If x is a Keyword or a Symbol, returns (name x),
otherwise returns (.toString x).

       (call ns f args)

counts

  function

  
Usage: (counts coll)


  
Returns a map with the items of coll as keys and the number
of occurrences of the item in coll as the corresponding values.

  Source


  

  

  
flatten-map

  function

  
Usage: (flatten-map m)


  
'Flattens' the map into a seq.
Each key contributes one of more elements to the seq. The elements
corresponding to any given key are obtained by calling pairs on
the to-seq of its value.

  Source


  

  

  
key-with-max-val

  function

  
Usage: (key-with-max-val map & [on-all-equal])


  
Returns the key which has the max value.
Returns on-all-equal if all the values are equal.

  Source


  

  

  
largest-n-by-vals

  function

  
Usage: (largest-n-by-vals n map)


  
Returns a subset of map containing the n largest entries
as determined by comparing the vals.

  Source


  

  

  
map-on-firsts

  function

  
Usage: (map-on-firsts f seq)


  
Takes a sequence of sequences (v1 v2 ...) and returns a lazy
sequence of ((f v1) (f v2) ...). If a collection is given, calls seq
on it.

  Source


  

  

  
map-on-lasts

  function

  
Usage: (map-on-lasts f pair-seq)


  
Takes a sequence of 2-item vectors [v1 v2] and returns a lazy
sequence of [v1 (f v2)]. If a collection is given, calls seq on it.

  Source


  

  

  
map-with-index

  function

  
Usage: (map-with-index f & colls)


  
Calls map on the collections after cons-ing (iterate inc 0) to them.
So f will receive the index along with the items themselves.

  Source


  

  

  
merge-with-+

  function

  
Usage: (merge-with-+ & maps)


  
Calls merge-with on maps with + as the function.

  Source


  

  

  
pairs

  function

  
Usage: (pairs key vals)


  
Returns a seq of vectors where each vector is a pairing of key with
a value from vals.

  Source


  

  

  
partition-random

  function

  
Usage: (partition-random n coll)


  
Randomly divides the items in coll into n partitions.
Returns a vector of vectors.

  Source


  

  

  
to-probabilities

  function

  
Usage: (to-probabilities coll)


  
Given a coll of numbers, converts them to probabilities
by dividing each with their total sum. Returns a seq.

  Source


  

  

  
to-seq

  function

  
Usage: (to-seq v)


  
Returns a seq.
If v implements Sequential, simply calls seq on it.  Otherwise returns
a seq containing v as the only element.
